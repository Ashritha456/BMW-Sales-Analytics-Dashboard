# BMW Worldwide Sales Analytics Dashboard

## Project Overview
This project analyzes BMW worldwide sales records from 2010 to 2024 using Power BI. The dashboard provides insights into global sales trends, regional performance, product-line performance, year-over-year growth, and market-level business KPIs.

## Dataset
Dataset Source: Kaggle  
Dataset Link: https://www.kaggle.com/datasets/ahmadrazakashif/bmw-worldwide-sales-records-20102024

The dataset contains BMW worldwide sales records from 2010 to 2024, including sales volume, regional performance, market trends, and product-line statistics.

## Business Questions
- Which regions contribute the most to BMW sales?
- How have BMW sales changed from 2010 to 2024?
- Which product lines show the strongest performance?
- What are the year-over-year growth patterns?
- How did global events such as COVID-19 affect sales trends?
